# PHP সিনট্যাক্স, আউটপুট (Printing) ও কমেন্টস

PHP একটি শক্তিশালী সার্ভার-সাইড স্ক্রিপ্টিং ভাষা, যেখানে নির্দিষ্ট কিছু নিয়ম (syntax) অনুসরণ করে কোড লিখতে হয়। এই নিয়মগুলো ভালোভাবে বুঝতে পারলে PHP শেখা অনেক সহজ হয়ে যায়। নিচে PHP-এর সবচেয়ে গুরুত্বপূর্ণ সিনট্যাক্সগুলো সহজভাবে ব্যাখ্যা করা হলো

## PHP সিনট্যাক্সের গুরুত্বপূর্ণ বিষয়সমূহ

| সিনট্যাক্স | বর্ণনা |
| ------ | ----------- |
| `<?php` এবং `?>` | এগুলো PHP কোড শুরু ও শেষ করার ট্যাগ। সব PHP কোড এই ট্যাগের মধ্যে লিখতে হয়। সার্ভার শুধুমাত্র এই অংশটুকুই PHP হিসেবে প্রসেস করে। |
| `//` | এটি এক লাইনের কমেন্ট লেখার জন্য ব্যবহার করা হয়। // এর পরে যা লেখা হয় তা কোড হিসেবে রান হয় না, শুধুমাত্র ডেভেলপারের বোঝার জন্য থাকে। |
| `/*` ... `*/` | এটি মাল্টি-লাইন কমেন্ট লেখার জন্য ব্যবহৃত হয়। একাধিক লাইনের ব্যাখ্যা বা নোট লেখার ক্ষেত্রে এটি ব্যবহার করা হয়। |
| `$variable` | PHP-তে ভেরিয়েবল সবসময় $ চিহ্ন দিয়ে শুরু হয়। এটি ডেটা সংরক্ষণ করার জন্য ব্যবহৃত হয়, যেমন নাম, সংখ্যা বা অন্যান্য তথ্য। |
| `=` | এটি অ্যাসাইনমেন্ট অপারেটর। কোনো ভেরিয়েবলের মধ্যে মান সংরক্ষণ করতে এটি ব্যবহার করা হয়। |
| `echo` | এটি সবচেয়ে বেশি ব্যবহৃত আউটপুট স্টেটমেন্ট। ওয়েব পেজে টেক্সট বা ভেরিয়েবলের মান দেখানোর জন্য ব্যবহার করা হয়। |
| `print` | echo-এর মতোই কাজ করে, তবে এটি তুলনামূলকভাবে কম ব্যবহৃত হয়। এটি একবারে একটি আউটপুট প্রদর্শন করতে পারে। |
| `if`, `else if`, `else` | এগুলো কন্ডিশনাল স্টেটমেন্ট। কোনো শর্ত সত্য বা মিথ্যা অনুযায়ী ভিন্ন ভিন্ন কোড চালানোর জন্য ব্যবহার করা হয়। |
| `while` | এটি একটি লুপ স্টেটমেন্ট। যতক্ষণ পর্যন্ত শর্ত সত্য থাকবে, ততক্ষণ কোড বারবার চলতে থাকবে। |
| `for` | নির্দিষ্ট সংখ্যক বার কোড চালানোর জন্য ব্যবহৃত হয়। সাধারণত কাউন্টিং বা নির্দিষ্ট রেঞ্জের ক্ষেত্রে ব্যবহার করা হয়। |
| `foreach` | অ্যারে বা অবজেক্টের প্রতিটি উপাদান একে একে প্রসেস করার জন্য ব্যবহৃত হয়। |
| `function` | এটি ফাংশন ডিফাইন করার জন্য ব্যবহৃত হয়। ফাংশন হলো পুনঃব্যবহারযোগ্য কোড ব্লক, যা প্রয়োজন অনুযায়ী বারবার ব্যবহার করা যায়। |
| `return` | ফাংশনের ভিতর থেকে কোনো মান ফেরত পাঠানোর জন্য ব্যবহার করা হয়। এটি ফাংশনের ফলাফল বাইরে পাঠায়। |

এগুলো PHP-এর সবচেয়ে সাধারণ কিছু সিনট্যাক্স এলিমেন্ট মাত্র। PHP-তে আরও অনেক ধরনের সিনট্যাক্স রয়েছে, যেগুলো বিভিন্ন উন্নত ফিচার ও জটিল কাজের জন্য ব্যবহার করা হয়। এছাড়া PHP একটি ক্রমাগত উন্নয়নশীল (constantly evolving) ভাষা, তাই নিয়মিতভাবে এতে নতুন নতুন ফিচার ও আপডেট যুক্ত হচ্ছে, যা এটিকে আরও শক্তিশালী ও আধুনিক করে তুলছে।

## PHP Comments (কমেন্টস)

PHP-তে কমেন্ট ব্যবহার করা হয় কোড ব্যাখ্যা বা নোট লেখার জন্য। এগুলো রান হয় না।

### Single Line Comment (এক লাইনের কমেন্ট)

```php
<?php
// এটি একটি এক লাইনের কমেন্ট
?>

```

### Multi Line Comment (একাধিক লাইনের কমেন্ট)

```php
<?php
/*
এটি একটি মাল্টি-লাইন কমেন্ট
এখানে একাধিক লাইনে ব্যাখ্যা লেখা যায়
*/
?>

```

## PHP Printing / Output (ডাটা দেখানো)

PHP-তে echo এবং print দুটিরই মূল কাজ হলো ব্রাউজারে আউটপুট দেখানো। প্রথম দেখায় এদের কাজ একই মনে হলেও, টেকনিক্যাল দিক থেকে এদের মধ্যে কিছু গুরুত্বপূর্ণ পার্থক্য আছে। নিচে এদের বিস্তারিত ব্যাখ্যা দেওয়া হলো:

### ১. echo স্টেটমেন্ট
echo হলো পিএইচপি-র একটি ল্যাঙ্গুয়েজ কনস্ট্রাক্ট (Language Construct), ফাংশন নয়। এটি আউটপুট দেওয়ার জন্য সবচেয়ে বেশি জনপ্রিয়।

**➤ মাল্টিপল প্যারামিটার:** echo ব্যবহার করে আপনি কমা (,) দিয়ে একাধিক স্ট্রিং একসাথে প্রিন্ট করতে পারেন। যেমন:
```php
<?php
echo "PHP ", "is ", "Fun!";
?>
```
**➤ রিটার্ন ভ্যালু:** এর কোনো রিটার্ন ভ্যালু নেই। অর্থাৎ, আপনি একে কোনো ভেরিয়েবলের মধ্যে স্টোর করতে পারবেন না।
**➤ গতি:** এটি print-এর চেয়ে সামান্য দ্রুত কাজ করে কারণ এটি কোনো ভ্যালু রিটার্ন করে না।
**➤ ব্যবহার:** সাধারণত সাধারণ টেক্সট, এইচটিএমএল ট্যাগ বা ভেরিয়েবল আউটপুট দিতে এটি ব্যবহার করা হয়।

### ২. print স্টেটমেন্ট
print-ও একটি ল্যাঙ্গুয়েজ কনস্ট্রাক্ট, তবে এটি কিছুটা ফাংশনের মতো আচরণ করে।

**➤ রিটার্ন ভ্যালু (সবচেয়ে বড় পার্থক্য):** print সবসময় একটি ইন্টিজার ভ্যালু হিসেবে 1 রিটার্ন করে। এর মানে হলো আপনি একে এক্সপ্রেশন হিসেবে ব্যবহার করতে পারেন।
```php
<?php
$status = print "Hello World";
?>
```
**➤ একটি মাত্র প্যারামিটার:** এটি শুধুমাত্র একটি আর্গুমেন্ট গ্রহণ করে। echo-এর মতো কমা দিয়ে অনেকগুলো স্ট্রিং একসাথে দেওয়া যায় না।
**➤ গতি:** এটি echo-এর চেয়ে কিছুটা ধীরগতির (খুবই সামান্য যা সাধারণ চোখে ধরা পড়ে না), কারণ এটি কাজ শেষে '1' রিটার্ন করার প্রসেসটি সম্পন্ন করে।

### এক নজরে পার্থক্য (Comparison Table)
| বৈশিষ্ট্য | echo | print |
| ------ | ----------- | ----------- |
| রিটার্ন ভ্যালু | কোনো ভ্যালু রিটার্ন করে না। | সবসময় 1 রিটার্ন করে। |
| প্যারামিটার | একাধিক প্যারামিটার নিতে পারে (যেমন: echo $a, $b;)। | শুধুমাত্র একটি প্যারামিটার নিতে পারে। |
| গতি | তুলনামূলক দ্রুত। | সামান্য ধীর। |
| ব্যবহারের ধরণ | ল্যাঙ্গুয়েজ কনস্ট্রাক্ট। | ল্যাঙ্গুয়েজ কনস্ট্রাক্ট (তবে এক্সপ্রেশনের মতো কাজ করে)। |

### কোনটি ব্যবহার করা ভালো?
একজন ডেভেলপার হিসেবে আপনি echo ব্যবহার করতেই স্বাচ্ছন্দ্যবোধ করবেন কারণ:

```➤ এটি দ্রুত।

➤ এটি ফ্লেক্সিবল (একাধিক ডাটা একসাথে কমা দিয়ে লেখা যায়)।

➤ পিএইচপি কমিউনিটিতে এটিই স্ট্যান্ডার্ড হিসেবে ধরা হয়।```

print তখনই দরকার হয় যদি আপনার আউটপুট দেওয়ার পাশাপাশি কোনো কন্ডিশন চেক করতে হয় যে আউটপুটটি সফলভাবে হয়েছে কি না (যদিও বাস্তবে এর প্রয়োজন খুব কম পড়ে)।

```নোট: যদি শুধু আউটপুট দেখাতে চাও তবে echo ব্যবহার করো, আর যদি এমন কোনো জায়গায় কোড লেখো যেখানে একটি রিটার্ন ভ্যালু দরকার, তখন print ব্যবহার করা যেতে পারে। ```

## PHP code

```php
<?php
/* নিচের কোডটি PHP-র বেসিক 
   আউটপুট এবং কমেন্ট প্রদর্শন করছে 
*/

// ভেরিয়েবল ডিক্লেয়ার করা
$language = "PHP"; // $language ভেরিয়েবলে একটি String (স্ট্রিং) মান অ্যাসাইন করা হয়েছে
$version = 8.2; // $version ভেরিয়েবলে একটি Float (ফ্লোট) মান অ্যাসাইন করা হয়েছে

// echo ব্যবহার করে আউটপুট
echo "Welcome to $language Tutorial! <br>"; // <br> ব্যবহার করা হয়েছে নতুন লাইনের জন্য

# print ব্যবহার করে আউটপুট
print "Current Version: " . $version; // ডট (.) দিয়ে স্ট্রিং জোড়া লাগানো হয়েছে
?>
```

## ব্যাখ্যা

এই কোডটি মূলত তিনটি প্রধান কাজ করছে: কমেন্ট, ভেরিয়েবল তৈরি, এবং আউটপুট দেখানো।

**১. কমেন্ট (Comments)** — কোডের নোট রাখা

কোডে কমেন্ট হলো এমন কিছু লেখা যা কম্পিউটার পড়ে না, শুধু আমরা (ডেভেলপাররা) বোঝার জন্য লিখি। এটি প্রোগ্রাম রান করার সময় কোনো প্রভাব ফেলে না।

➤ সিঙ্গেল লাইন কমেন্ট (// বা #): যখন শুধু এক লাইনের জন্য কিছু লিখতে চাই, তখন লাইনের শুরুতে // বা # ব্যবহার করি। যেমন: // ভেরিয়েবল ডিক্লেয়ার করা
➤ মাল্টি-লাইন কমেন্ট (/* ... */): যদি অনেক বড় করে বা কয়েক লাইন জুড়ে নোট লিখতে চাই, তখন এটি ব্যবহার করি। যেমন:
```php
/* নিচের কোডটি PHP-র বেসিক 
   আউটপুট এবং কমেন্ট প্রদর্শন করছে 
*/
```

**২. ভেরিয়েবল (Variables)** — তথ্য জমা রাখা

PHP-তে ভেরিয়েবল হলো একটি ছোট পাত্রের মতো যেখানে আমরা তথ্য জমা রাখতে পারি।

➤ নিয়ম: ভেরিয়েবল সবসময় ডলার চিহ্ন ($) দিয়ে শুরু করতে হয়।

➤ কোড অনুযায়ী: এখানে $language ভেরিয়েবলে আমরা "PHP" এবং $version ভেরিয়েবলে 8.2 মানটি জমা রেখেছি।

**৩. আউটপুট (echo ও print)** — তথ্য প্রদর্শন করা

জমা রাখা তথ্যগুলো ব্রাউজারে বা স্ক্রিনে দেখানোর জন্য আমরা echo বা print ব্যবহার করি।

➤ echo: এটি ব্যবহার করে আমরা "Welcome to PHP Tutorial!" প্রিন্ট করেছি। echo-তে ভেরিয়েবলকে সরাসরি ডাবল কোটেশনের ভেতরে লিখলে সেটির মান দেখায়।

➤ print: এটি দিয়ে আমরা ভার্সনটি প্রিন্ট করেছি।

➤ ডট (.) এর কাজ: পিএইচপি-তে দুটি আলাদা লেখাকে বা টেক্সটের সাথে ভেরিয়েবলকে জোড়া দিতে ডট (.) চিহ্ন ব্যবহার করা হয় (যাকে বলা হয় Concatenation)।
