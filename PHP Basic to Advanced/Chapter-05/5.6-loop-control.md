# অধ্যায় ৫: লুপ এবং ইটারেসন (PHP Loops & Iteration)


## ৫.৬ Loop Control (break / continue)

PHP তে `break` এবং `continue` ব্যবহার করা হয় Loop এর flow বা execution control করার জন্য।

অনেক সময় Loop চলাকালীন এমন পরিস্থিতি আসে যেখানে:

- Loop সম্পূর্ণ বন্ধ করতে হয়
- কিছু নির্দিষ্ট iteration skip করতে হয়
- নির্দিষ্ট condition এ loop terminate করতে হয়

এই ধরনের পরিস্থিতি handle করার জন্য `break` এবং `continue` ব্যবহার করা হয়।


### Break Statement
---

`break` ব্যবহার করা হয় Loop সম্পূর্ণ বন্ধ করার জন্য।

যখন কোনো condition true হয় এবং `break` execute হয়, তখন loop সাথে সাথে বন্ধ হয়ে যায় এবং loop এর পরের code execute হতে শুরু করে।



### Syntax

```php
<?php
    for ($i = 1; $i <= 10; $i++) {

        if ($i == 5) {

            break;
        }

        echo $i . "<br>";
    }
?>
```


### Example 1: পণ্য অনুসন্ধান সিস্টেম (Product Search System)
---

বাস্তব প্রজেক্টে Product Search করার সময় যখন কাঙ্ক্ষিত Product পাওয়া যায়, তখন আর পুরো list search করার প্রয়োজন হয় না।

এই ধরনের ক্ষেত্রে `break` ব্যবহার করা হয়।

```php
<?php

    $products = [
        "Laptop",
        "Keyboard",
        "Mouse",
        "Monitor"
    ];

    foreach ($products as $product) {

        if ($product == "Mouse") {

            echo "Product Found: $product <br>";

            break;
        }

        echo "Searching Product: $product <br>";
    }
?>
```



### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- Loop প্রথমে `Laptop` access করবে
- এরপর `Keyboard`
- তারপর `Mouse`
- যখন `Mouse` পাওয়া যাবে, তখন condition true হবে
- `break` execute হওয়ার সাথে সাথে loop বন্ধ হয়ে যাবে
- এরপর আর `Monitor` check হবে না



### Output

```text
Searching Product: Laptop
Searching Product: Keyboard
Product Found: Mouse
```



### Example 2: অর্ডার প্রসেসিং স্টপ সিস্টেম (Order Processing Stop System)
---

ধরুন, একটি Order Processing System এ যদি কোনো Failed Order পাওয়া যায়, তাহলে পুরো Processing বন্ধ করে দিতে হবে।

```php
<?php
    $orders = [
        "Completed",
        "Completed",
        "Failed",
        "Completed"
    ];

    foreach ($orders as $orderStatus) {

        if ($orderStatus == "Failed") {

            echo "Order Processing Stopped! Failed Order Detected.<br>";

            break;
        }

        echo "Order Processed Successfully.<br>";
    }
?>
```



### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- প্রথম দুইটি Order Successfully Process হবে
- তৃতীয় Order এ `Failed` পাওয়া যাবে
- Condition true হওয়ার পর `break` execute হবে
- Loop সাথে সাথে বন্ধ হয়ে যাবে
- পরের Order আর Process হবে না



### Output

```text
Order Processed Successfully.
Order Processed Successfully.
Order Processing Stopped! Failed Order Detected.
```



### Continue Statement
---

`continue` ব্যবহার করা হয় Loop এর নির্দিষ্ট iteration skip করার জন্য।

যখন `continue` execute হয়, তখন loop এর বর্তমান iteration এর বাকি code skip হয়ে যায় এবং loop সরাসরি পরবর্তী iteration এ চলে যায়।



#### Syntax

```php
<?php
    for ($i = 1; $i <= 5; $i++) {

        if ($i == 3) {
            continue;
        }

        echo $i . "<br>";
    }
?>
```



### Example 3: স্টক নেই এমন পণ্য এড়িয়ে যাওয়ার সিস্টেম (Out of Stock Product Skip System)
---

বাস্তব E-commerce Project এ অনেক সময় কিছু Product Out of Stock থাকে।

এই ধরনের Product skip করে শুধুমাত্র Available Product display করার জন্য `continue` ব্যবহার করা হয়।

```php
<?php

    $products = [
        [
            "name" => "Laptop",
            "stock" => true
        ],

        [
            "name" => "Mouse",
            "stock" => false
        ],

        [
            "name" => "Keyboard",
            "stock" => true
        ]
    ];

    foreach ($products as $product) {

        if ($product['stock'] == false) {

            continue;

        }

        echo "Available Product: " . $product['name'] . "<br>";
    }
?>
```



### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- প্রথম Product `Laptop` display হবে
- দ্বিতীয় Product `Mouse` এর stock false
- তাই `continue` execute হবে
- বর্তমান iteration এর বাকি code skip হবে
- Loop সরাসরি পরবর্তী Product এ যাবে
- এরপর `Keyboard` display হবে



### Output

```text
Available Product: Laptop
Available Product: Keyboard
```


### Example 4: ছাত্র উপস্থিতি এড়ানোর সিস্টেম (Student Attendance Skip System)
---

ধরুন, একটি Student Attendance System এ যারা অনুপস্থিত তাদের skip করে শুধুমাত্র উপস্থিত ছাত্রদের display করতে হবে।

```php
<?php

    $students = [
        [
            "name" => "Rahim",
            "present" => true
        ],

        [
            "name" => "Karim",
            "present" => false
        ],

        [
            "name" => "Jakir",
            "present" => true
        ]
    ];

    foreach ($students as $student) {

        if ($student['present'] == false) {
            continue;
        }

        echo "Present Student: " . $student['name'] . "<br>";
    }
?>
```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- প্রথম Student `Rahim` display হবে
- দ্বিতীয় Student `Karim` absent
- তাই `continue` execute হবে
- Karim এর iteration skip হয়ে যাবে
- এরপর `Jakir` display হবে



### Output

```text
Present Student: Rahim
Present Student: Jakir
```



### ⚠️ গুরুত্বপূর্ণ বিষয়

| Statement | কাজ |
|---|---|
| `break` | পুরো loop বন্ধ করে দেয় |
| `continue` | শুধুমাত্র current iteration skip করে |

### 🎯 কেন Loop Control গুরুত্বপূর্ণ?

এটি ব্যবহার করে:

- Search Optimization করা যায়
- Failed Process Stop করা যায়
- Invalid Data Skip করা যায়
- Filtering System তৈরি করা যায়
- Performance Improve করা যায়
- Smart Iteration Control করা যায়



### 📚 সারসংক্ষেপ

> `break` এবং `continue` হলো PHP এর Loop Control Statement। `break` loop সম্পূর্ণ terminate করে দেয়, আর `continue` current iteration skip করে loop কে পরবর্তী iteration এ নিয়ে যায়।
