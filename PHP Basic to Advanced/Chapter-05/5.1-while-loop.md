# অধ্যায় ৫: লুপ এবং ইটারেসন (PHP Loops & Iteration)

## ৫.১ While Loop

PHP তে `while` loop ব্যবহার করা হয় একটি নির্দিষ্ট শর্ত (Condition) সত্য থাকা পর্যন্ত কোনো কোড ব্লককে বারবার রান বা এক্সিকিউট করার জন্য।

প্রোগ্রামিং করার সময় অনেক ক্ষেত্রে এমন পরিস্থিতি আসে যেখানে লুপটি ঠিক কতবার ঘুরবে তা আমরা আগে থেকে নিশ্চিতভাবে জানি না, তবে একটি নির্দিষ্ট শর্ত পূরণ না হওয়া পর্যন্ত কাজটি চালিয়ে যাওয়া প্রয়োজন হয়। এই পরিস্থিতিগুলোর সঠিক ও কার্যকর ব্যবস্থাপনার জন্য `while` loop ব্যবহার করা হয়।


### while Statement
---

`while` loop মূলত কোনো নির্দিষ্ট কন্ডিশন বা শর্ত শুরুতেই সত্য কিনা তা পরীক্ষা বা যাচাই করে। 

যদি কন্ডিশন সত্য (True) হয়, তবেই এটি ব্লকের ভেতরের কোড রান করবে। প্রতিবার কোড রান করার পর এটি আবার কন্ডিশন চেক করে। যতক্ষণ পর্যন্ত শর্তটি সত্য থাকবে, লুপটি চলতেই থাকবে। যখনই শর্তটি মিথ্যা (False) হয়ে যাবে, লুপটি সাথে সাথে বন্ধ হয়ে যাবে। শর্ত শুরুতেই মিথ্যা হলে এই লুপ একবারও চলে না।

### Syntax

```php
initialization; // লুপের শুরুর মান নির্ধারণ

while (condition) { // শর্ত যাচাই
    // condition সত্য হলে এই কোড বারবার চলবে
    increment/decrement; // লুপ কন্ট্রোল ভ্যারিয়েবলের মান পরিবর্তন
}
```

### Example 1: একটি নির্দিষ্ট সংখ্যা পর্যন্ত লুপ চালানো (Running Loop Up to a Certain Number)
---

আসুন একটি বাস্তব উদাহরণ কোডের মাধ্যমে দেখি। মনে করুন, একটি খালি বালতিতে ১ লিটার করে পানি ঢেলে ৫ লিটার পূর্ণ করার লজিক আমরা তৈরি করব।

```php
    <?php
        $waterLevel = 1; // initialization: শুরুতে বালতিতে ১ লিটার পানি আছে

        while ($waterLevel <= 5) { // condition: যতক্ষণ পানি ৫ লিটার বা তার কম
            echo "বালতিতে আরও এক লিটার পানি ঢালুন। বর্তমান পানি: $waterLevel লিটার। <br>";
            
            $waterLevel++; // increment: প্রতিবার ১ লিটার করে পানি বাড়ছে
        }
    ?>
```

### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:
- **শুরুর ধাপ:** প্রথমে `$waterLevel` এর মান ১ সেট করা হয়েছে (১ম লিটার)
- **শর্ত যাচাই:** লুপ চেক করবে `১ কি ৫ এর চেয়ে ছোট বা সমান?` হ্যাঁ, সত্য (True)। তাই ভেতরের কোড রান করে স্ক্রিনে আউটপুট দেখাবে
- **মান পরিবর্তন:** `$waterLevel++` এর মাধ্যমে পানি ১ লিটার বৃদ্ধি পেয়ে মান হবে ২
- **পুনরাবৃত্তি:** লুপটি একইভাবে ২, ৩, ৪, এবং ৫ পর্যন্ত ঘুরবে
- **লুপ সমাপ্তি:** যখনই মান বেড়ে ৬ হবে, তখন শর্তটি `(৬ <= ৫)` মিথ্যা (False) হয়ে যাবে এবং লুপটি বন্ধ হয়ে যাবে

### Output

```text
বালতিতে আরও এক লিটার পানি ঢালুন। বর্তমান পানি: ১ লিটার।
বালতিতে আরও এক লিটার পানি ঢালুন। বর্তমান পানি: ২ লিটার।
বালতিতে আরও এক লিটার পানি ঢালুন। বর্তমান পানি: ৩ লিটার।
বালতিতে আরও এক লিটার পানি ঢালুন। বর্তমান পানি: ৪ লিটার।
বালতিতে আরও এক লিটার পানি ঢালুন। বর্তমান পানি: ৫ লিটার।
```

### Example 2: Array-এর উপাদান প্রিন্ট করা (Printing Array Elements)
---

বাস্তব লাইফে আমাদের অনেক সময় কোনো লিস্ট বা অ্যারে (Array) থেকে একে একে ডেটা বের করে দেখানোর প্রয়োজন হয়। নিচে `while` loop ব্যবহার করে একটি অ্যারের ভেতর থেকে সব বন্ধুদের নাম প্রিন্ট করার উদাহরণ দেওয়া হলো:

```php
<?php

    $index = 0; // initialization: কাউন্টার শুরু হচ্ছে ০ থেকে (যেহেতু অ্যারের ইনডেক্স ০ থেকে শুরু হয়)
    $names = ['Karim', 'Rahim', 'Abul', 'Hasan', 'Latif']; // আমাদের বন্ধুদের নামের একটি অ্যারে

    while ($index < count($names)) { // condition: যতক্ষণ ইনডেক্স অ্যারের মোট উপাদানের চেয়ে ছোট থাকবে
        echo "বন্ধুর নাম: " . $names[$index] . "<br>";
        
        $index++; // increment: প্রতিবার ইনডেক্সের মান ১ করে বাড়বে
    }

?>
```

### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:
- **`count($names)` কী করে:** এই ফাংশনটি অ্যারের ভেতরে মোট কতটি উপাদান আছে তা গুনে বের করে। আমাদের এই অ্যারেতে ৫টি নাম আছে, তাই `count($names)` এর রিটার্ন মান হবে ৫
- **শর্ত যাচাই (Condition):** লুপের কন্ডিশনে বলা হয়েছে `$index < 5`। অর্থাৎ ইনডেক্সের মান ৫ এর চেয়ে ছোট হওয়া পর্যন্ত লুপটি চলবে (ইনডেক্স নম্বর: ০, ১, ২, ৩, এবং ৪)
- **ধাপে ধাপে কার্যপ্রবাহ:**
    - **১ম বার:** `$index = 0` (যা ৫ এর চেয়ে ছোট)। স্ক্রিনে প্রিন্ট হবে `$names[0]` অর্থাৎ `Karim` এবং ইনডেক্স বেড়ে হবে ১
    - **২য় বার:** `$index = 1` (যা ৫ এর চেয়ে ছোট)। স্ক্রিনে প্রিন্ট হবে `$names[1]` অর্থাৎ `Rahim` এবং ইনডেক্স বেড়ে হবে ২
    - **৩য় বার:** `$index = 2` (যা ৫ এর চেয়ে ছোট)। স্ক্রিনে প্রিন্ট হবে `$names[2]` অর্থাৎ `Abul` এবং ইনডেক্স বেড়ে হবে ৩
    - **৪র্থ বার:** `$index = 3` (যা ৫ এর চেয়ে ছোট)। স্ক্রিনে প্রিন্ট হবে `$names[3]` অর্থাৎ `Hasan` এবং ইনডেক্স বেড়ে হবে ৪
    - **৫ম বার:** `$index = 4` (যা ৫ এর চেয়ে ছোট)। স্ক্রিনে প্রিন্ট হবে `$names[4]` অর্থাৎ `Latif` এবং ইনডেক্স বেড়ে হবে ৫
- **লুপ সমাপ্তি:** যখনই মান বেড়ে ৫ হবে, তখন শর্তটি `(৫ < ৫)` মিথ্যা (False) হয়ে যাবে এবং লুপটি স্বয়ংক্রিয়ভাবে বন্ধ হয়ে যাবে

### Output

```text
বন্ধুর নাম: Karim
বন্ধুর নাম: Rahim
বন্ধুর নাম: Abul
বন্ধুর নাম: Hasan
বন্ধুর নাম: Latif
```

### ⚠️ সতর্কতা (Infinite Loop):

লুপের ভেতরে যদি মান পরিবর্তনের ধাপ বা `increment/decrement` করতে ভুলে যান, তবে কন্ডিশনটি আজীবনের জন্য সত্য রয়ে যাবে। এর ফলে লুপটি কখনো বন্ধ হবে না, যাকে Infinite Loop বলা হয়। এটি আপনার ব্রাউজার বা সার্ভার ক্র্যাশ করাতে পারে।


### 🎯 কেন while loop গুরুত্বপূর্ণ?

এটি ব্যবহার করে:

- নির্দিষ্ট শর্তের উপর ভিত্তি করে পুনরাবৃত্তিমূলক কাজ করা যায়
- ডেটাবেসের রেকর্ড বা ফাইল থেকে অজানা সংখ্যক ডেটা শেষ না হওয়া পর্যন্ত রিড করা যায়
- ডায়নামিক ও শর্তসাপেক্ষ গেম বা অ্যাপ্লিকেশন লজিক তৈরি করা যায়
- ইউজার সঠিক ইনপুট না দেওয়া পর্যন্ত পুনরায় ইনপুট চাওয়ার মেকানিজম তৈরি করা যায়


### 📚 সারসংক্ষেপ

> `while loop` হলো পিএইচপির একটি এন্ট্রি-কন্ট্রোলড (Entry-controlled) লুপ, যা লুপের শুরুতে শর্ত যাচাই করে এবং শর্তটি সত্য থাকা পর্যন্ত নির্দিষ্ট কোড ব্লককে পুনরাবৃত্তি করতে সাহায্য করে।
