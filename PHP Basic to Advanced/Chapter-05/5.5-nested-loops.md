# অধ্যায় ৫: লুপ এবং ইটারেসন (PHP Loops & Iteration)

## ৫.৫ Nested Loop

PHP তে `Nested Loop` বলতে বোঝায় একটি Loop এর ভেতরে আরেকটি Loop ব্যবহার করা।

প্রোগ্রামিংয়ের সময় অনেক ক্ষেত্রে এমন পরিস্থিতি আসে যেখানে একটি ডাটার ভেতরে আরও একাধিক ডাটা থাকে। যেমন:

- Product এর ভেতরে Multiple Variant
- Student এর ভেতরে Multiple Subject
- Order এর ভেতরে Multiple Product
- Table Row এবং Column তৈরি করা

এই ধরনের Complex Iteration পরিচালনা করার জন্য `Nested Loop` ব্যবহার করা হয়।



### Nested Loop Statement

যখন একটি Loop এর ভিতরে আরেকটি Loop থাকে, তখন বাইরের Loop প্রতিবার execute হওয়ার সময় ভিতরের Loop সম্পূর্ণভাবে execute হয়।

অর্থাৎ:

- Outer Loop একবার চললে
- Inner Loop সম্পূর্ণ একবার চলবে

তারপর Outer Loop আবার পরবর্তী iteration এ যাবে।



### Syntax

```php
<?php

    for (initialization; condition; increment/decrement) {

        // Outer Loop Code

        for (initialization; condition; increment/decrement) {

            // Inner Loop Code

        }
    }
?>
```



### Example 1: পণ্য ভ্যারিয়েন্ট ব্যবস্থাপনা সিস্টেম (Product Variant Management System)
---

বাস্তব প্রজেক্টে একটি Product এর একাধিক Variant থাকতে পারে।

যেমন:

- T-Shirt
    - Red
    - Blue
    - Black

- Shoe
    - 40 Size
    - 41 Size
    - 42 Size

এই ধরনের data display করার জন্য Nested Loop ব্যবহার করা হয়।

```php
<?php

    $products = [
        "T-Shirt" => ["Red", "Blue", "Black"],
        "Shoe" => ["40 Size", "41 Size", "42 Size"]
    ];

    foreach ($products as $productName => $variants) {

        echo "<h3>$productName</h3>";

        foreach ($variants as $variant) {

            echo "- $variant <br>";

        }

        echo "<br>";
    }
?>
```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- `$products` একটি Multidimensional Associative Array
- বাইরের `foreach` loop প্রতিবার একটি Product access করবে
- `$productName` এ Product Name থাকবে
- `$variants` এ Product এর Variant List থাকবে
- ভিতরের `foreach` loop প্রতিটি Variant access করবে
- প্রতিটি Product এর নিচে তার Variant গুলো display হবে
- সব Product এবং Variant শেষ হলে loop বন্ধ হয়ে যাবে



### Output

```text
T-Shirt
- Red
- Blue
- Black

Shoe
- 40 Size
- 41 Size
- 42 Size
```


### Example 2: শিক্ষার্থী বিষয়ভিত্তিক ফলাফল ব্যবস্থা (Student Subject Result System)
---

ধরুন, একটি School Management System এ প্রতিটি Student এর একাধিক Subject এর Result রয়েছে।

এই ধরনের Nested Data display করার জন্য Nested Loop ব্যবহার করা হয়।

```php
<?php

    $students = [
        "Rahim" => [
            "Math"    => 85,
            "English" => 78,
            "ICT"     => 90
        ],

        "Karim" => [
            "Math"    => 88,
            "English" => 82,
            "ICT"     => 95
        ]
    ];

    foreach ($students as $studentName => $subjects) {

        echo "<h3>Student: $studentName</h3>";

        foreach ($subjects as $subject => $mark) {

            echo "$subject : $mark <br>";

        }

        echo "<br>";
    }
?>
```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- `$students` একটি Multidimensional Associative Array
- বাইরের Loop প্রতিবার একটি Student access করছে
- `$studentName` এ Student এর নাম আসছে
- `$subjects` এ সেই Student এর Subject List আসছে
- ভিতরের Loop প্রতিটি Subject এবং Mark access করছে
- প্রতিটি Student এর Result আলাদা করে display হচ্ছে
- সব Student এর data শেষ হলে loop বন্ধ হয়ে যাবে



### Output

```text
Student: Rahim
Math : 85
English : 78
ICT : 90

Student: Karim
Math : 88
English : 82
ICT : 95
```


### Example 3: অর্ডার এবং পণ্য ব্যবস্থাপনা সিস্টেম (Order and Product Management System)
---

বাস্তব E-commerce Project এ একটি Order এর মধ্যে একাধিক Product থাকে।

যেমন:

- Order #1001
    - Laptop
    - Mouse

- Order #1002
    - Keyboard
    - Monitor

এই ধরনের data display করার জন্য Nested Loop ব্যবহার করা হয়।

```php
<?php

    $orders = [
        "ORD-1001" => [
            "Laptop",
            "Mouse"
        ],

        "ORD-1002" => [
            "Keyboard",
            "Monitor"
        ]
    ];

    foreach ($orders as $orderId => $products) {

        echo "<h3>Order ID: $orderId</h3>";

        foreach ($products as $product) {
            echo "- $product <br>";
        }

        echo "<br>";
    }
?>
```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- `$orders` একটি Multidimensional Array
- বাইরের Loop প্রতিবার একটি Order access করছে
- `$orderId` এ Order ID রাখা হচ্ছে
- `$products` এ সেই Order এর Product List রাখা হচ্ছে
- ভিতরের Loop প্রতিটি Product access করছে
- প্রতিটি Order এর Product গুলো এক এক করে display হচ্ছে
- সব Order শেষ হলে loop automatically বন্ধ হয়ে যাবে



### Output

```text
Order ID: ORD-1001
- Laptop
- Mouse

Order ID: ORD-1002
- Keyboard
- Monitor
```



### ⚠️ সতর্কতা (Performance Issue)

Nested Loop অতিরিক্ত ব্যবহার করলে Performance সমস্যা হতে পারে।

কারণ:

- Outer Loop যদি 100 বার চলে
- Inner Loop যদি 100 বার চলে

তাহলে মোট iteration হবে:

```text
100 x 100 = 10,000
```

তাই Large Data নিয়ে কাজ করার সময় Nested Loop সতর্কতার সাথে ব্যবহার করতে হবে।



### 🎯 কেন Nested Loop গুরুত্বপূর্ণ?

এটি ব্যবহার করে:

- Multidimensional Array Handle করা যায়
- Table/Data Grid তৈরি করা যায়
- Product Variant System তৈরি করা যায়
- Order & Item Management করা যায়
- Student Result System তৈরি করা যায়
- Complex Data Structure Process করা যায়
