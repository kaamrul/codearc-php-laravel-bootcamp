# অধ্যায় ৫: লুপ এবং ইটারেসন (PHP Loops & Iteration)

## ৫.৩ For Loop

PHP তে `for loop` ব্যবহার করা হয় এমন কাজগুলো বারবার করার জন্য যেখানে আমরা আগে থেকেই জানি লুপটি কতবার চলবে। এটি প্রোগ্রামিংয়ে সবচেয়ে বেশি ব্যবহৃত লুপ।

প্রোগ্রামিংয়ের সময় অনেক ক্ষেত্রে নির্দিষ্ট সংখ্যক iteration বা পুনরাবৃত্তি প্রয়োজন হয়। যেমন: ১ থেকে ১০০ পর্যন্ত সংখ্যা প্রিন্ট করা, ৫০ জন ছাত্রের রোল দেখানো, বা নির্দিষ্ট সংখ্যক ডাটা প্রসেস করা। এই ধরনের কাজগুলো খুব সহজ ও গুছানোভাবে করার জন্য `for loop` ব্যবহার করা হয়।


### For Loop Statement

`for loop` মূলত তিনটি অংশ নিয়ে কাজ করে, এখানে এক লাইনের মধ্যেই ৩টি জিনিস থাকে:

1. **Initialization** → লুপ কোথা থেকে শুরু হবে
2. **Condition** → কতক্ষণ পর্যন্ত চলবে
3. **Increment/Decrement** → প্রতিবার iteration শেষে মান কীভাবে পরিবর্তন হবে

প্রথমে initialization একবার execute হয়। এরপর condition check করা হয়। condition যদি সত্য (True) হয়, তাহলে loop এর ভেতরের code execute হয়। প্রতিবার code execute হওয়ার পর increment/decrement অংশ কাজ করে এবং আবার condition check হয়।

যখন condition মিথ্যা (False) হয়ে যায়, তখন loop বন্ধ হয়ে যায়।


## Syntax

```php
<?php

    for (initialization; condition; increment/decrement) {

        // code to be executed
    }

?>
```


### Example 1: 

আসুন একটি বাস্তব উদাহরণ দেখি। মনে করুন, একটি স্কুল ম্যানেজমেন্ট সফটওয়্যারে ১ থেকে ৫ পর্যন্ত ছাত্রদের রোল নম্বর দেখানো হবে।

```php

<?php

    for ($roll = 1; $roll <= 5; $roll++) {

        echo "Student Roll Number: $roll <br>";

    }
?>

```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:
- **শুরুর ধাপ:** `$roll = 1` দিয়ে loop শুরু হবে
- **শর্ত যাচাই:** এরপর চেক করবে `$roll <= 5` কি সত্য?
- **কোড রান:** শর্ত সত্য হলে রোল নম্বর প্রিন্ট করবে
- **মান পরিবর্তন:** `$roll++` এর মাধ্যমে মান ১ করে বাড়বে
- **পুনরাবৃত্তি:** একইভাবে loop ১ থেকে ৫ পর্যন্ত চলবে
- **লুপ সমাপ্তি:** `$roll` এর মান ৬ হবে, condition false হয়ে loop বন্ধ হবে

### Output

```text
Student Roll Number: 1
Student Roll Number: 2
Student Roll Number: 3
Student Roll Number: 4
Student Roll Number: 5
```



### Example 1: ই-কমার্স অর্ডার প্রসেসিং সিস্টেম (E-commerce Order Processing System)

বাস্তব প্রজেক্টে আমরা প্রায়ই এমন পরিস্থিতির মুখোমুখি হই যেখানে একটি নির্দিষ্ট সংখ্যক অর্ডার প্রসেস করতে হয়।

ধরুন, একটি E-commerce Website এ আজকে মোট ৫টি নতুন অর্ডার এসেছে। এখন সিস্টেম প্রতিটি অর্ডার এক এক করে প্রসেস করবে এবং status update করবে।

যেহেতু আমরা আগে থেকেই জানি মোট ৫টি অর্ডার আছে, তাই এখানে `for loop` ব্যবহার করাই সবচেয়ে উপযুক্ত।

```php

<?php

    $totalOrders = 5;

    for ($order = 1; $order <= $totalOrders; $order++) {

        echo "অর্ডার নম্বর $order প্রসেস করা হচ্ছে... <br>";

        echo "অর্ডার নম্বর $order সফলভাবে সম্পন্ন হয়েছে। <br><br>";

    }
?>

```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:
- **শুরুর ধাপ:** `$order = 1` দিয়ে loop শুরু হবে
- **শর্ত যাচাই:** এরপর চেক করবে `$order <= 5` কি সত্য?
- **কোড রান:** শর্ত সত্য হলে রোল নম্বর প্রিন্ট করবে
- **মান পরিবর্তন:** `$order++` এর মাধ্যমে মান ১ করে বাড়বে
- **পুনরাবৃত্তি:** একইভাবে loop ১ থেকে ৫ পর্যন্ত চলবে
- **লুপ সমাপ্তি:** `$order` এর মান ৬ হবে, condition false হয়ে loop বন্ধ হবে

### Output

```text

অর্ডার নম্বর 1 সফলভাবে সম্পন্ন হয়েছে।

অর্ডার নম্বর 2 প্রসেস করা হচ্ছে...

অর্ডার নম্বর 3 প্রসেস করা হচ্ছে...
অর্ডার নম্বর 3 সফলভাবে সম্পন্ন হয়েছে।

অর্ডার নম্বর 4 প্রসেস করা হচ্ছে...

```


### ⚠️ সতর্কতা (Infinite Loop):

লুপের ভেতরে যদি মান পরিবর্তনের ধাপ বা `increment/decrement` করতে ভুলে যান, তবে কন্ডিশনটি আজীবনের জন্য সত্য রয়ে যাবে। এর ফলে লুপটি কখনো বন্ধ হবে না, যাকে Infinite Loop বলা হয়। এটি আপনার ব্রাউজার বা সার্ভার ক্র্যাশ করাতে পারে।

```php
<?php

    for ($i = 1; $i <= 5;) {

        echo $i;

    }
?>
```

> এখানে $i এর মান কখনো পরিবর্তন হচ্ছে না, তাই loop অনন্তকাল চলবে।




### 🎯 কেন For Loop গুরুত্বপূর্ণ?

এটি ব্যবহার করে:

- নির্দিষ্ট সংখ্যক কাজ বারবার করা যায়
- Large Data Processing করা যায়
- Table/Data Generate করা যায়
- Pagination Logic তৈরি করা যায়
- Report Generation System তৈরি করা যায়
- Batch Processing করা যায়

### 📚 সারসংক্ষেপ

> `for loop` হলো পিএইচপির একটি Entry-controlled Loop, যা নির্দিষ্ট সংখ্যক iteration এর জন্য সবচেয়ে বেশি ব্যবহৃত হয়। এটি initialization, condition এবং increment/decrement — এই তিনটি অংশ নিয়ে কাজ করে।
