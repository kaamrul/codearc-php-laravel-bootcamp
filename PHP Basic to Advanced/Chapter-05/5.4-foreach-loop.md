# অধ্যায় ৫: লুপ এবং ইটারেসন (PHP Loops & Iteration)

## ৫.৪ Foreach Loop

PHP তে `foreach loop` ব্যবহার করা হয় Array বা Collection এর প্রতিটি ডাটা সহজভাবে এক এক করে access করার জন্য।

প্রোগ্রামিংয়ের সময় আমরা প্রায়ই একাধিক ডাটা নিয়ে কাজ করি। যেমন:

- Student List
- Product List
- User Data
- Database Records

এই ধরনের ডাটা সাধারণত Array আকারে থাকে। Array এর প্রতিটি element এক এক করে access এবং process করার জন্য `foreach loop` সবচেয়ে বেশি ব্যবহৃত হয়।

`foreach loop` এর সবচেয়ে বড় সুবিধা হলো এখানে manually counter increase/decrease করতে হয় না। Loop নিজে থেকেই Array এর প্রতিটি item নিয়ে কাজ করে।



### Foreach Loop Statement

`foreach loop` মূলত Array এর প্রতিটি value এক এক করে নিয়ে কাজ করে।

Loop শুরু হলে Array এর প্রথম element access হয়। এরপর প্রতিবার iteration এ পরবর্তী element access হয়। Array এর সবগুলো data শেষ হয়ে গেলে loop automatically বন্ধ হয়ে যায়।

এটি সাধারণত দুইভাবে ব্যবহার করা হয়:

1. শুধুমাত্র Value Access করার জন্য
2. Key এবং Value দুটোই Access করার জন্য



### Syntax (Only Value)

```php
<?php

    foreach ($array as $value) {

        // code to be executed

    }

?>
```

### Syntax (Key & Value)

```php
<?php

    foreach ($array as $key => $value) {

        // code to be executed

    }

?>
```


### Example 1: পণ্যের বিবরণ (Product Details)
---

বাস্তব প্রজেক্টে একটি E-commerce Website এ প্রতিটি Product এর বিভিন্ন তথ্য থাকে। যেমন:

- Product Name
- Category
- Price
- Quantity
- Status
- Variant

এই ধরনের data সাধারণত Associative Array আকারে রাখা হয় এবং `foreach loop` ব্যবহার করে এক এক করে display করা হয়।

```php
<?php

    $productDetails = [
        "name"     => "Gaming Laptop",
        "category" => "Electronics",
        "price"    => 85000,
        "quantity" => 12,
        "status"   => "In Stock",
        "variant"  => "16GB RAM / 512GB SSD"
    ];

    foreach ($productDetails as $key => $value) {

        echo ucfirst($key) . " : " . $value . "<br>";

    }
?>
```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- `$productDetails` একটি Associative Array যেখানে Product এর বিভিন্ন তথ্য রাখা হয়েছে
- `foreach ($productDetails as $key => $value)` এর মাধ্যমে প্রতিবার একটি key এবং তার value access করা হচ্ছে
- `$key` variable এ field name (name, category, price) আসবে
- `$value` variable এ সেই field এর value আসবে
- `ucfirst()` function ব্যবহার করে key এর প্রথম অক্ষর বড় করে দেখানো হয়েছে
- Array এর সব data শেষ হলে loop automatically বন্ধ হয়ে যাবে


### Output

```text
Name : Gaming Laptop
Category : Electronics
Price : 85000
Quantity : 12
Status : In Stock
Variant : 16GB RAM / 512GB SSD
```




### Example 2: পণ্যের তালিকা প্রদর্শন (Product List Display)
---

ধরুন, একটি Online Shop এ একাধিক Product এর নাম, price এবং quantity দেখানো হবে।

এই ধরনের ক্ষেত্রে Multidimensional Array এবং `foreach loop` ব্যবহার করা হয়।

```php
<?php

    $products = [

        [
            "name"     => "Laptop",
            "price"    => 85000,
            "quantity" => 5
        ],

        [
            "name"     => "Keyboard",
            "price"    => 2500,
            "quantity" => 20
        ],

        [
            "name"     => "Mouse",
            "price"    => 1200,
            "quantity" => 15
        ]

    ];

    foreach ($products as $product) {

        echo "Product Name : " . $product['name'] . "<br>";

        echo "Price : " . $product['price'] . " Tk <br>";

        echo "Quantity : " . $product['quantity'] . "<br><br>";

    }

?>
```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- `$products` একটি Multidimensional Array যেখানে একাধিক Product রাখা হয়েছে
- প্রতিটি Product আবার একটি Associative Array
- বাইরের `foreach` loop প্রতিবার একটি Product access করছে
- `$product['name']` দিয়ে Product Name access করা হচ্ছে
- `$product['price']` দিয়ে Price access করা হচ্ছে
- `$product['quantity']` দিয়ে Quantity access করা হচ্ছে
- প্রতিবার loop একটি Product এর সম্পূর্ণ তথ্য display করবে
- সব Product শেষ হলে loop automatically বন্ধ হয়ে যাবে


### Output

```text
Product Name : Laptop
Price : 85000 Tk
Quantity : 5

Product Name : Keyboard
Price : 2500 Tk
Quantity : 20

Product Name : Mouse
Price : 1200 Tk
Quantity : 15
```


### Example 3: অর্ডার তালিকা (Order List)
---

অর্ডার তালিকাতে প্রতিটি Order এর বিভিন্ন তথ্য থাকে। যেমন:

- Order ID
- Customer Name
- Quantity
- Purchase Price
- Order Status

এই ধরনের Order List সাধারণত Database থেকে আসে এবং foreach loop ব্যবহার করে এক এক করে display করা হয়।

```php
<?php

    $orders = [

        [
            "order_id"       => "ORD-1001",
            "customer_name" => "Rahim",
            "quantity"      => 2,
            "purchase_price"=> 2500,
            "order_status"  => "Pending"
        ],

        [
            "order_id"       => "ORD-1002",
            "customer_name" => "Karim",
            "quantity"      => 1,
            "purchase_price"=> 85000,
            "order_status"  => "Completed"
        ],

        [
            "order_id"       => "ORD-1003",
            "customer_name" => "Jannat",
            "quantity"      => 3,
            "purchase_price"=> 4500,
            "order_status"  => "Processing"
        ]

    ];

    foreach ($orders as $order) {

        echo "Order ID : " . $order['order_id'] . "<br>";

        echo "Customer Name : " . $order['customer_name'] . "<br>";

        echo "Quantity : " . $order['quantity'] . "<br>";

        echo "Purchase Price : " . $order['purchase_price'] . " Tk <br>";

        echo "Order Status : " . $order['order_status'] . "<br><br>";

    }

?>
```


### এক্সাম্পল ডিটেইলস ও যেভাবে কাজ করবে:

- `$orders` একটি Multidimensional Associative Array যেখানে একাধিক Order এর তথ্য রাখা হয়েছে
- প্রতিটি Order এর মধ্যে Order ID, Customer Name, Quantity, Purchase Price এবং Status রয়েছে
- `foreach ($orders as $order)` প্রতিবার একটি Order access করছে
- `$order['order_id']` দিয়ে Order ID নেওয়া হচ্ছে
- `$order['customer_name']` দিয়ে Customer Name নেওয়া হচ্ছে
- `$order['quantity']` দিয়ে Quantity নেওয়া হচ্ছে
- `$order['purchase_price']` দিয়ে Purchase Price নেওয়া হচ্ছে
- `$order['order_status']` দিয়ে বর্তমান Order Status নেওয়া হচ্ছে
- প্রতিবার loop একটি Order এর সম্পূর্ণ তথ্য display করবে
- সব Order শেষ হলে loop automatically বন্ধ হয়ে যাবে


### Output

```text
Order ID : ORD-1001
Customer Name : Rahim
Quantity : 2
Purchase Price : 2500 Tk
Order Status : Pending

Order ID : ORD-1002
Customer Name : Karim
Quantity : 1
Purchase Price : 85000 Tk
Order Status : Completed

Order ID : ORD-1003
Customer Name : Jannat
Quantity : 3
Purchase Price : 4500 Tk
Order Status : Processing
```



### ⚠️ সতর্কতা (Infinite Loop):

`foreach loop` শুধুমাত্র Array এবং Collection এর উপর কাজ করে।

যদি সাধারণ variable এর উপর `foreach` ব্যবহার করেন, তাহলে error দেখাবে।

X ভুল Example:

```php
<?php

    $name = "Kamrul";

    foreach ($name as $value) {

        echo $value;

    }
?>
```




### 🎯 কেন Foreach Loop গুরুত্বপূর্ণ?

এটি ব্যবহার করে:

- Database Records Display করা যায়
- Product List দেখানো যায়
- User Data Access করা যায়
- API Response Process করা যায়
- Dynamic Table তৈরি করা যায়
- Menu/List Generate করা যায়
