# অধ্যায় ৩: অপারেটর এবং এক্সপ্রেশন (Operators & Expressions)

## ৩.১ গাণিতিক বা অ্যারিথমেটিক অপারেটর (Arithmetic Operators)

গাণিতিক হিসাব-নিকাশের জন্য পিএইচপি বেশ কিছু অপারেটর প্রদান করে। এর মধ্যে যোগ (+), বিয়োগ (-), গুণ (*), ভাগ (/) ছাড়াও গুরুত্বপূর্ণ দুটি অপারেটর হলো মডুলাস (%) এবং এক্সপোনেনশিয়েসন () ।

- মডুলাস (%): এটি ভাগশেষ নির্ণয় করে। লজিক বিল্ডিংয়ে যেমন জোড়-বিজোড় সংখ্যা নির্ণয়ে এটি অপরিহার্য।
- এক্সপোনেনশিয়েসন (): এটি কোনো সংখ্যার পাওয়ার বা ঘাত নির্ণয় করে (যেমন: 2^3 = 8)।


## Arithmetic Operators Table

| অপারেটর | নাম | উদাহরণ | ফলাফল বর্ণনা |
|---|---|---|---|
| `+` | যোগ (Addition) | `$a + $b` | অপারেন্ড দুটির সমষ্টি |
| `-` | বিয়োগ (Subtraction) | `$a - $b` | প্রথম অপারেন্ড থেকে দ্বিতীয়টির বিয়োগফল |
| `*` | গুণ (Multiplication) | `$a * $b` | অপারেন্ড দুটির গুণফল |
| `/` | ভাগ (Division) | `$a / $b` | ভাগফল (ফলাফল ফ্লোট হতে পারে) |
| `%` | মডুলাস (Modulus) | `$a % $b` | ভাগ করার পর অবশিষ্ট বা ভাগশেষ |
| `**` | এক্সপোনেনশিয়েসন (Exponentiation) | `$a ** $b` | ঘাত বা পাওয়ার (যেমন 2^3 = 8)  |


## 🧠 উদাহরণ (Example)

```php
<?php

// ========================================
// PHP Arithmetic Operators Example
// এই কোডে সকল arithmetic operator দেখানো হয়েছে
// ========================================

$a = 10;
$b = 3;

// Addition
echo $a + $b; // 13

// Subtraction
echo $a - $b; // 7

// Multiplication
echo $a * $b; // 30

// Division
echo $a / $b; // 3.3333

// Modulus
echo $a % $b; // 1

// Exponentiation
echo $a ** $b; // 1000
?>
```
