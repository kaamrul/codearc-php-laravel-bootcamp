# অধ্যায় ৩: অপারেটর এবং এক্সপ্রেশন (Operators & Expressions)

## ৩.২ ইনক্রিমেন্ট ও ডিক্রিমেন্ট অপারেটর (Increment & Decrement)

ভ্যারিয়েবলের মান ১ করে বাড়ানোর জন্য ++ এবং কমানোর জন্য -- ব্যবহৃত হয়। তবে এর দুটি রূপ রয়েছে: Pre (পূর্ব) এবং Post (পরবর্তী) ।

- **++$count (Pre-increment):** মান আগে বাড়ে, তারপর এক্সপ্রেশনে ব্যবহৃত হয়।
- **$count++ (Post-increment):** বর্তমান মান আগে ব্যবহৃত হয়, তারপর বৃদ্ধি পায়।


## Increment / Decrement Operators Table

| অপারেটর | নাম | ধরন | উদাহরণ | ফলাফল বর্ণনা |
|---|---|---|---|---|
| `++` | ইনক্রিমেন্ট অপারেটর (Increment) | Pre-increment | `++$a` | আগে মান ১ বাড়ে, তারপর ব্যবহার হয় |
| `++` | ইনক্রিমেন্ট অপারেটর (Increment) | Post-increment | `$a++` | আগে বর্তমান মান ব্যবহার হয়, তারপর ১ বাড়ে |
| `--` | ডিক্রিমেন্ট অপারেটর (Decrement) | Pre-decrement | `--$a` | আগে মান ১ কমে, তারপর ব্যবহার হয় |
| `--` | ডিক্রিমেন্ট অপারেটর (Decrement) | Post-decrement | `$a--` | আগে বর্তমান মান ব্যবহার হয়, তারপর ১ কমে |


## উদাহরণ (Example)

```php
<?php

// ========================================
// PHP Increment & Decrement Operators
// Professional Example with explanation
// ========================================

$a = 10;

// Post-increment
echo $a++; // Output: 10 (আগে ব্যবহার, পরে বৃদ্ধি)

// এখন $a = 11

// Pre-increment
echo ++$a; // Output: 12 (আগে বৃদ্ধি, পরে ব্যবহার)

// Decrement example
echo $a--; // Output: 12 (আগে ব্যবহার, পরে কমে 11)

// Final value
echo $a;   // Output: 11

?>
```

## Section A: MCQ (Multiple Choice Questions)

### 1. PHP তে increment operator কোনটি?
A) `--`
B) `++`
C) `+=`
D) `**`  

✔ Answer: B) `++`

---

### 2. `$a++` কোন ধরনের increment?
A) Pre-increment
B) Post-increment
C) Pre-decrement
D) Post-decrement 

✔ Answer: B) Post-increment

---

### 3. `++$a` এর অর্থ কী?
A) আগে ব্যবহার, পরে বাড়ে
B) আগে বাড়ে, পরে ব্যবহার হয়
C) কমে যায়
D) কোন পরিবর্তন হয় না

✔ Answer: B) আগে বাড়ে, পরে ব্যবহার হয়

---

### 4. Decrement operator কোনটি?
A) `++`
B) `--`
C) `//`
D) `**`

✔ Answer: B) `--`

---

### 5. `$a--` কী করে?

A) মান ২ বাড়ায়
B) মান ১ কমায়
C) মান ১ বাড়ায়
D) কিছুই করে না

✔ Answer: B) মান ১ কমায়

---

### 6. Pre-increment এর সঠিক উদাহরণ কোনটি?

A) `$a++`
B) `$a--`
C) `++$a`
D) `$a =+ 1`

✔ Answer: C) `++$a`

---

### 7. Post-decrement এর সঠিক ব্যবহার কোনটি?

A) `--$a`
B) `$a--`
C) `++$a`
D) `$a++`

✔ Answer: B) `$a--`

---

### 8. Increment operator কত যোগ করে?

A) 2
B) 5
C) 1
D) 0

✔ Answer: C) 1

## Section C: Short Answer Questions

### 1. Increment operator কী?
👉 উত্তর: Increment operator (++) কোনো variable এর মান ১ বাড়ায়।

---

### 2. Pre-increment এবং Post-increment এর পার্থক্য কী?
👉 উত্তর:

- Pre-increment (++$a) আগে মান বাড়ায়, তারপর ব্যবহার করে
- Post-increment ($a++) আগে ব্যবহার করে, তারপর মান বাড়ায় 

---

### 3. Decrement operator কী?
👉 উত্তর: Decrement operator (`--`) কোনো variable এর মান ১ কমায়।

---

### 4. উদাহরণ দাও Post-increment এর।

👉 উত্তর:

```php
<?php
$a = 5;
echo $a++; // আগে 5 দেখাবে, পরে 6 হবে
?>
```
---

### 5. উদাহরণ দাও Pre-increment এর।

👉 উত্তর:

```php
<?php
$a = 5;
echo ++$a; // আগে 6 হবে, তারপর print হবে
?>
```
