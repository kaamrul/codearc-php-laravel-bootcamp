# অধ্যায় ৪: পিএইচপি কন্ট্রোল স্ট্রাকচার (PHP Control Structure)

## ৪.৬ ব্রেক এবং কন্টিনিউ (Break & Continue)

PHP তে `break` এবং `continue` ব্যবহার করা হয় loop এর flow control করার জন্য।

এগুলো ব্যবহার করে loop এর execution থামানো বা নির্দিষ্ট iteration skip করা যায়।



### Break Statement
---

`break` ব্যবহার করা হয় loop বা switch statement সম্পূর্ণভাবে বন্ধ করার জন্য।



### Syntax

```php
break;
```


### Example

```php
<?php
    for ($i = 1; $i <= 10; $i++) {
        if ($i == 5) {
            break;
        }

        echo $i . " ";
    }
?>
```


### Output

```text
1 2 3 4
```


### Explanation

উপরের উদাহরণে `$i == 5` হলে loop বন্ধ হয়ে যায়।  
তাই 5 থেকে পরের সংখ্যা আর print হয় না।



### Continue Statement
---

`continue` ব্যবহার করা হয় বর্তমান iteration skip করে পরের iteration এ যাওয়ার জন্য।


### Syntax

```php
continue;
```


### Example

```php
<?php
    for ($i = 1; $i <= 10; $i++) {

        if ($i == 5) {
            continue;
        }

        echo $i . " ";
    }
?>
```



### Output

```text
1 2 3 4 6 7 8 9 10
```



### Explanation

উপরের উদাহরণে `$i == 5` হলে সেই iteration skip করা হয়।  
তাই 5 print হয় না, কিন্তু loop চলতে থাকে।



### Break vs Continue
---

| বিষয় | Break | Continue |
|------|------|----------|
| কাজ | loop বন্ধ করে দেয় | নির্দিষ্ট iteration skip করে |
| প্রভাব | পুরো loop stop | loop চলতে থাকে |
| ব্যবহার | condition পূরণ হলে loop থামানো | নির্দিষ্ট value ignore করা |



### Real Life Example
---

### Break Example: Search System

```php
<?php
    $users = ["Rahim", "Karim", "Jamal", "Kamal"];

    foreach ($users as $user) {
        if ($user == "Jamal") {
            echo "User found!";
            break;
        }
    }
?>
```

### Output
```text
User found!
```



### Continue Example: Skip Banned Users


```php
<?php
    $users = ["Rahim", "Banned", "Karim", "Banned", "Jamal"];

    foreach ($users as $user) {
        if ($user == "Banned") {
            continue;
        }

        echo $user . " ";
    }
?>
```

### Output
```text
Rahim Karim Jamal
```



## 🎯 কেন Break & Continue গুরুত্বপূর্ণ?

- Loop control করা যায়  
- Unwanted data skip করা যায়  
- Performance improve হয়  
- Search / filter system বানানো যায়  
- Complex logic সহজ হয়
