# অধ্যায় ৪: পিএইচপি কন্ট্রোল স্ট্রাকচার (PHP Control Structure)

## ৪.৪ টারনারি অপারেটর (Ternary Operator)

PHP তে **Ternary Operator** হলো `if...else` এর সংক্ষিপ্ত (short-hand) রূপ।

ছোট এবং সহজ condition এক লাইনে লেখার জন্য Ternary Operator ব্যবহার করা হয়। এটি কোডকে আরও concise এবং readable করতে সাহায্য করে।



### Ternary Operator
---

Ternary Operator মূলত তিনটি অংশ নিয়ে কাজ করে:

- একটি condition
- condition সত্য হলে কী হবে
- condition মিথ্যা হলে কী হবে



### Syntax

```php
condition ? true_value : false_value;
```



### Example

```php
<?php

    $age = 20;

    $message = ($age >= 18) ? "আপনি প্রাপ্তবয়স্ক" : "আপনি অপ্রাপ্তবয়স্ক";

    echo $message;

?>
```

### Output

```text
আপনি প্রাপ্তবয়স্ক
```



### if...else বনাম Ternary Operator
---

### Using if...else

```php
<?php

    $status = true;

    if ($status == true) {
        echo "Active";
    } else {
        echo "Inactive";
    }

?>
```



### Using Ternary Operator


```php
<?php

    $status = true;

    echo ($status == true) ? "Active" : "Inactive";

?>
```


### Real-Life Example: User Online Status
---

Ternary Operator বাস্তব জীবনের বিভিন্ন dynamic system এ ব্যবহার করা হয়।

যেমন:
- User Online/Offline Status
- Active/Inactive Account
- Payment Success/Failed Message
- Product Stock Availability
- Verification Status



### Example

```php
<?php

    $isOnline = true;

    $status = ($isOnline == true) ? "User Online" : "User Offline";

    echo $status;
?>
```

### Output

```text
User Online
```



### Explanation

উপরের উদাহরণে `$isOnline` variable এর condition যাচাই করা হয়েছে।

- যদি `true` হয় → `User Online`
- যদি `false` হয় → `User Offline`

এভাবেই খুব ছোট condition এক লাইনে লিখে দ্রুত decision নেওয়া যায়।



## 🎯 কেন Ternary Operator গুরুত্বপূর্ণ?

Ternary Operator ব্যবহার করে:

- ছোট condition সহজভাবে লেখা যায়
- Code concise হয়
- Readability বৃদ্ধি পায়
- Dynamic value assign করা যায়
- Simple decision making দ্রুত করা যায়



## ⚠️ গুরুত্বপূর্ণ বিষয়

- শুধুমাত্র ছোট এবং simple condition এর জন্য Ternary ব্যবহার করা ভালো
- অতিরিক্ত complex logic Ternary দিয়ে না লেখাই উত্তম
- Nested ternary code readability কমিয়ে দিতে পারে



## 📚 সারসংক্ষেপ

> Ternary Operator হলো `if...else` এর সংক্ষিপ্ত রূপ, যা ছোট condition এক লাইনে লিখে দ্রুত decision নিতে সাহায্য করে।
