# অধ্যায় ৪: পিএইচপি কন্ট্রোল স্ট্রাকচার (PHP Control Structure)

## ৪.৩ সুইচ স্টেটমেন্ট (Switch Statement)

PHP তে `switch` statement ব্যবহার করা হয় একটি নির্দিষ্ট expression বা variable এর বিভিন্ন সম্ভাব্য মান যাচাই করার জন্য।

যখন একই variable এর জন্য একাধিক condition যাচাই করার প্রয়োজন হয়, তখন বারবার `if...elseif` ব্যবহার করার পরিবর্তে `switch` ব্যবহার করা বেশি সহজ, পরিষ্কার এবং readable হয়।



### Switch Statement
---

`switch` statement একটি expression এর মান যাচাই করে এবং matching `case` অনুযায়ী নির্দিষ্ট কোড execute করে।



### Syntax

```php
switch (expression) {

    case value1:
        // value1 match হলে এই কোড চলবে
        break;

    case value2:
        // value2 match হলে এই কোড চলবে
        break;

    default:
        // কোনো case match না হলে এই কোড চলবে
}
```


### Example

```php
<?php

    $day = "Friday";

    switch ($day) {

        case "Saturday":
            echo "আজ ছুটির দিন";
            break;

        case "Friday":
            echo "আজ জুমার দিন";
            break;

        case "Sunday":
            echo "আজ অফিস খোলা";
            break;

        default:
            echo "সাধারণ কর্মদিবস";
    }

?>
```

### Output

```text
আজ জুমার দিন
```


### break Statement এর ব্যবহার

প্রতিটি `case` এর শেষে সাধারণত `break` ব্যবহার করা হয়।

`break` ব্যবহার না করলে নিচের case গুলোও execute হতে থাকবে।



### Example Without break
---

```php
<?php

    $number = 1;

    switch ($number) {

        case 1:
            echo "One ";
        
        case 2:
            echo "Two ";
        
        case 3:
            echo "Three ";
    }

?>
```

### Output

```text
One Two Three
```



### Real-Life Example: Role Permission System
---

Switch statement বাস্তব জীবনের Role Permission System তৈরিতে অনেক বেশি ব্যবহার করা হয়।

যেমন:
- Admin Dashboard Access
- Editor Permission
- User Panel Access
- Moderator Control System


### Example

```php
<?php

    $userRole = "editor";

    switch ($userRole) {

        case "admin":
            echo "Full Dashboard Access";
            break;

        case "editor":
            echo "Content Editing Access";
            break;

        case "moderator":
            echo "Comment Management Access";
            break;

        case "user":
            echo "General User Access";
            break;

        default:
            echo "No Access Found";
    }

?>
```

### Output

```text
Content Editing Access
```



### Explanation

উপরের উদাহরণে `$userRole` variable এর মান যাচাই করা হয়েছে।

- Role যদি `admin` হয় → Full Dashboard Access
- Role যদি `editor` হয় → Content Editing Access
- Role যদি `moderator` হয় → Comment Management Access
- Role যদি `user` হয় → General User Access
- কোনো role match না হলে → No Access Found

এভাবেই Switch Statement ব্যবহার করে সহজে Role-Based Access Control System তৈরি করা যায়।



## 🎯 কেন Switch Statement গুরুত্বপূর্ণ?

Switch ব্যবহার করে:

- Multiple condition সহজভাবে handle করা যায়
- Code readability বৃদ্ধি পায়
- বড় if...elseif chain এড়ানো যায়
- Cleaner decision-making system তৈরি করা যায়
- Role management system তৈরি করা যায়



## ⚠️ গুরুত্বপূর্ণ বিষয়

- প্রতিটি `case` এর শেষে `break` ব্যবহার করা ভালো practice
- `default` block optional হলেও ব্যবহার করা recommended
- একই variable এর multiple value check করার জন্য `switch` সবচেয়ে উপযোগী



## 📚 সারসংক্ষেপ

> Switch Statement ব্যবহার করে একটি variable বা expression এর বিভিন্ন সম্ভাব্য মান যাচাই করে নির্দিষ্ট condition অনুযায়ী আলাদা আলাদা কাজ সম্পাদন করা যায়।
