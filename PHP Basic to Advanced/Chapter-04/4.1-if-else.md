# অধ্যায় ৪: পিএইচপি কন্ট্রোল স্ট্রাকচার (PHP Control Structure)

## ৪.১ If / Else / Elseif

PHP তে `if`, `else`, এবং `elseif` ব্যবহার করা হয় condition বা শর্তের উপর ভিত্তি করে সিদ্ধান্ত নেওয়ার জন্য।

প্রোগ্রামিং করার সময় অনেক ক্ষেত্রে এমন পরিস্থিতি আসে যেখানে একটি শর্ত সত্য হলে এক ধরনের কাজ করতে হয় এবং মিথ্যা হলে অন্য ধরনের কাজ করতে হয়। এই decision making process পরিচালনার জন্য `if / else / elseif` ব্যবহার করা হয়।


### if Statement

`if` ব্যবহার করা হয় কোনো condition সত্য কিনা যাচাই করার জন্য।

যদি condition সত্য হয়, তাহলে `if` ব্লকের ভেতরের কোড execute হবে।

### Syntax

```php
if (condition) {
    // condition সত্য হলে এই কোড চলবে
}
```

### Example

```php
<?php

$age = 20;

if ($age >= 18) {
    echo "আপনি ভোট দিতে পারবেন";
}

?>
```

### Output

```text
আপনি ভোট দিতে পারবেন
```


### if...else Statement

যখন condition সত্য হলে এক ধরনের কাজ এবং মিথ্যা হলে অন্য কাজ করতে হয়, তখন `if...else` ব্যবহার করা হয়।

### Syntax

```php
if (condition) {
    // condition সত্য হলে চলবে
} else {
    // condition মিথ্যা হলে চলবে
}
```

### Example

```php
<?php

$age = 16;

if ($age >= 18) {
    echo "আপনি প্রাপ্তবয়স্ক";
} else {
    echo "আপনি অপ্রাপ্তবয়স্ক";
}

?>
```

### Output

```text
আপনি অপ্রাপ্তবয়স্ক
```


### if...elseif...else Statement

একাধিক condition যাচাই করার জন্য `elseif` ব্যবহার করা হয়।

### Syntax

```php
    if (condition1) {
        // condition1 সত্য হলে চলবে
    } elseif (condition2) {
        // condition2 সত্য হলে চলবে
    } else {
        // সব condition মিথ্যা হলে চলবে
    }
```

### Example

```php
<?php
    $marks = 75;

    if ($marks >= 80) {
        echo "Grade A+";
    } elseif ($marks >= 70) {
        echo "Grade A";
    } elseif ($marks >= 60) {
        echo "Grade A-";
    } else {
        echo "Fail";
    }
?>
```

### Output

```text
Grade A
```


## 🎯 কেন if / else / elseif গুরুত্বপূর্ণ?

এগুলো ব্যবহার করে:

- Decision making করা যায়
- Dynamic application তৈরি করা যায়
- User input অনুযায়ী আলাদা output দেখানো যায়
- Login system তৈরি করা যায়
- Validation করা যায়
- Role management তৈরি করা যায়



## 📚 সারসংক্ষেপ

> `if`, `else`, এবং `elseif` ব্যবহার করে condition অনুযায়ী প্রোগ্রামের flow নিয়ন্ত্রণ করা হয় এবং বিভিন্ন পরিস্থিতিতে ভিন্ন ভিন্ন কাজ সম্পাদন করা সম্ভব হয়।
