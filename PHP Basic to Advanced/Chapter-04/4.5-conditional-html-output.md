# অধ্যায় ৪: পিএইচপি কন্ট্রোল স্ট্রাকচার (PHP Control Structure)

## ৪.৫ কন্ডিশনাল HTML আউটপুট (Conditional HTML Output)

PHP তে condition অনুযায়ী HTML show বা hide করাকে **Conditional HTML Output** বলা হয়।

এর মাধ্যমে dynamic web page তৈরি করা যায়, যেখানে বিভিন্ন condition অনুযায়ী আলাদা HTML content user কে দেখানো হয়।



### Conditional HTML Output
---

PHP এর `if`, `else`, `elseif`, এবং `ternary operator` ব্যবহার করে condition অনুযায়ী HTML render করা যায়।

এটি modern web application development এর একটি গুরুত্বপূর্ণ অংশ।



### Syntax

```php
<?php if(condition): ?>

    <!-- condition সত্য হলে এই HTML দেখাবে -->

<?php else: ?>

    <!-- condition মিথ্যা হলে এই HTML দেখাবে -->

<?php endif; ?>
```



### Example

```php
<?php

$isLoggedIn = true;

?>

<!DOCTYPE html>
<html>
<head>
    <title>Conditional HTML</title>
</head>
<body>

<?php if($isLoggedIn): ?>

    <h1>Welcome Back User</h1>

<?php else: ?>

    <h1>Please Login First</h1>

<?php endif; ?>

</body>
</html>
```



### Output

```text
Welcome Back User
```


### Using Ternary Operator in HTML
---

Conditional HTML Output এ Ternary Operator ও ব্যবহার করা যায়।

### Example

```php
<?php

$isPremium = true;

?>

<h1>
    <?= ($isPremium) ? "Premium User" : "Free User"; ?>
</h1>
```



### Output

```text
Premium User
```



### Real-Life Example: Role-Based Navigation Menu
---

Conditional HTML Output বাস্তব জীবনের অনেক web application এ ব্যবহার করা হয়।

যেমন:
- Login/Register Button
- Admin Dashboard Menu
- Premium Content Access
- Role-Based Navigation
- User Verification Badge



### Example

```php
<?php

$userRole = "admin";

?>

<ul>

    <li>Home</li>
    <li>Profile</li>

    <?php if($userRole == "admin"): ?>
        <li>Admin Dashboard</li>
        <li>User Management</li>
    <?php endif; ?>

</ul>
```



### Output

```html
Home
Profile
Admin Dashboard
User Management
```



### Explanation

উপরের উদাহরণে `$userRole` যদি `admin` হয়, তাহলে অতিরিক্ত menu item দেখানো হবে।

অন্য user হলে শুধুমাত্র সাধারণ menu গুলো দেখাবে।

এভাবেই condition অনুযায়ী dynamically HTML control করা যায়।



## 🎯 কেন Conditional HTML Output গুরুত্বপূর্ণ?

Conditional HTML Output ব্যবহার করে:

- Dynamic UI তৈরি করা যায়
- User Role অনুযায়ী content দেখানো যায়
- Secure navigation system তৈরি করা যায়
- Personalized user experience দেওয়া যায়
- Modern web application তৈরি করা যায়



## ⚠️ গুরুত্বপূর্ণ বিষয়

- PHP condition এবং HTML structure পরিষ্কারভাবে maintain করা উচিত
- অতিরিক্ত nested condition HTML readability কমিয়ে দিতে পারে
- Sensitive content show করার আগে permission check করা গুরুত্বপূর্ণ
